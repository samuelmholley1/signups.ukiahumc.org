# UI Demo Pages

## Card Layout Demo
**URL:** `signups.ukiahumc.org/demo-cards`

**Description:** Alternate UI design for food distribution signups using vertical card layout instead of horizontal table rows.

**Features:**
- Vertical card layout (2 cards per date, side by side)
- December 2025 Saturdays
- 2 volunteer slots
- Modal signup form
- Green theme

**Use Case:** Reference design if we want to try different UI approaches in the future.
